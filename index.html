Quick Guide

Deleting Objects

Double-click (desktop) or long-press (mobile) on any object or line to delete it. No confirmation needed — use undo to recover.

Animation Playback

Click Play to watch lines animate sequentially in the order they were drawn. A dot traces each path, then the line appears. Click Stop to halt and restore. No tools work during playback.

Tags & Categories

Below the drill description, type a tag name and press Enter. Tags double as categories in the Library. A drill tagged "Skating" and "Warmup" appears in both category sections. Remove a tag by clicking its × button.

Share via Link

Click Share to copy a URL containing the entire drill. Send it to another coach — opening the link imports the drill into their library automatically, tagged "Shared".

Practice Plans

The Practice Plan tab lets you build a session from your library drills. Set a duration and notes for each entry, reorder with arrows, and export as a multi-page PDF with an overview table plus one page per drill.

Print Cards

In the Library tab, click Print Cards to generate a PDF with drill cards — choose 1, 2, or 4 cards per page using the layout selector. Each card shows a rink thumbnail, title, tags, and description. Optionally enter a team name to add a header with team name and date on each page. Handy for printing and bringing to the rink.

Keyboard Shortcuts

ESC — deselect tool  |  Ctrl/Cmd+Z — undo  |  Ctrl/Cmd+Y — redo

Data & Storage

Everything is saved in your browser's local storage — no accounts, no cloud. Use Export Library to back up all drills as one JSON file. Use Import Library to restore (merge or replace). Individual drills can also be exported/imported as JSON.

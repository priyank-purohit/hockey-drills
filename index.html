Quick Guide

Drawing Lines

Select a line tool (Skate, Puck, Pass, Shoot), then click once on the rink to set the start point, and click again to set the end point. The arrow appears at the end. A preview line follows your cursor between clicks.

Placing & Moving Objects

Select a player, net, pylon, or puck tool, then click on the rink to place it. To reposition, select the Move tool and drag any object. To delete, double-click (desktop) or long-press (mobile) on any object.

Text Annotations

Select the A Text tool. A text input appears below the toolbar — type your annotation, then click on the rink to place it. The input stays active so you can place multiple labels.

Rink Views

The Full / Left / Right / NZ buttons zoom into different zones. Your drill data is always stored in full-rink coordinates — switching views just changes what's visible. Objects placed in one view remain when you switch.

Animation Playback

Click Play to watch lines animate sequentially in the order they were drawn. A dot traces each path, then the line appears. Click Stop to halt and restore. No tools work during playback.

Tags & Categories

Below the drill description, type a tag name and press Enter. Tags double as categories in the Library. A drill tagged "Skating" and "Warmup" appears in both category sections. Remove a tag by clicking its × button.

Share via Link

Click Share to copy a URL containing the entire drill. Send it to another coach — opening the link imports the drill into their library automatically, tagged "Shared".

Practice Plans

The Practice Plan tab lets you build a session from your library drills. Set a duration and notes for each entry, reorder with arrows, and export as a multi-page PDF with an overview table plus one page per drill.

Print Cards

In the Library tab, click Print Cards to generate a PDF with 2 drill cards per page — each card shows a rink thumbnail, title, tags, and description. Handy for printing and bringing to the rink.

Keyboard Shortcuts

ESC — deselect tool  |  Ctrl/Cmd+Z — undo  |  Ctrl/Cmd+Y — redo

Data & Storage

Everything is saved in your browser's local storage — no accounts, no cloud. Use Export Library to back up all drills as one JSON file. Use Import Library to restore (merge or replace). Individual drills can also be exported/imported as JSON.

# Text Annotations

## Implemented
- [x] Text tool in toolbar (Annotate section)
- [x] Text input bar appears when tool selected
- [x] Click on rink to place annotation
- [x] Font size picker — small / medium / large
- [x] Rotation selector — 0 / 45 / 90 / -45 / -90 degrees
- [x] Double-click to edit text after placement
- [x] Colour override support
- [x] Serializes text, size, rotation in drill data
- [x] Input stays active for placing multiple labels

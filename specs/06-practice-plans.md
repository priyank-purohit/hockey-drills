# Practice Plans

## Implemented
- [x] Practice Plan tab
- [x] Multiple practice plans with selector dropdown
- [x] New Plan / Delete Plan buttons
- [x] Add drills from library to plan
- [x] Non-drill entries (Water Break, Free Skate, etc.)
- [x] Duration per entry (minutes)
- [x] Notes per entry
- [x] Reorder with up/down arrows
- [x] Remove entries
- [x] Total duration in clock format (Xh Xm)
- [x] Export as multi-page landscape PDF
  - Page 1: overview table
  - Subsequent pages: one per drill with title, duration, notes, description
  - Break entries in overview only (no detail page)
- [x] Plan persists in library localStorage

# Move Tool

## Implemented
- [x] Move button in toolbar
- [x] Objects become draggable when Move tool active
- [x] Blue dashed highlight on movable objects
- [x] Grab/grabbing cursor on hover/drag
- [x] Logical coordinates auto-update on drop
- [x] Works with undo/redo
- [x] Lines movable as groups (start/end coords update)
- [x] Move mode re-enabled after canvas re-renders

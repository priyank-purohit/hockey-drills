# Sharing

## Share Drill
- [x] Share button generates URL with drill in hash fragment
- [x] LZ-String compression for shorter URLs (~50-70% reduction)
- [x] URL copied to clipboard with visual feedback
- [x] QR code modal for phone scanning
- [x] Backward compatible with old base64 URLs
- [x] Recipients get drill auto-imported tagged "Shared"
- [x] Warning if URL exceeds 8000 characters

## Share Practice Plan
- [x] Share Plan button in Practice Plan tab
- [x] Encodes plan + all referenced drills
- [x] Uses #plan= URL prefix (vs #drill= for drills)
- [x] Recipients get drills imported + plan added
- [x] Auto-switches to Practice Plan tab on load
